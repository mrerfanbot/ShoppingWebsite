# 💄 لوکس‌بیوتی — فروشگاه لوازم آرایشی و بهداشتی

طراحی UI/UX با Bootstrap 5 RTL و فارسی

---

## 🖼️ صفحات UI

پنج صفحه کامل و Responsive در پوشه [`ui/`](./ui/) قرار دارند:

| صفحه | فایل | توضیح |
|------|------|--------|
| صفحه اصلی | [`ui/home.html`](./ui/home.html) | هدر، اسلایدر، دسته‌بندی‌ها، محصولات ویژه، برندها، فوتر |
| لیست محصولات | [`ui/products.html`](./ui/products.html) | فیلتر، گرید/لیست، مرتب‌سازی، صفحه‌بندی |
| جزئیات محصول | [`ui/product-detail.html`](./ui/product-detail.html) | گالری، قیمت، انتخاب‌گر، تب‌ها، نظرات، محصولات مشابه |
| سبد خرید | [`ui/cart.html`](./ui/cart.html) | لیست سبد، تعداد، کد تخفیف، خلاصه سفارش، مراحل checkout |
| داشبورد مدیریت | [`ui/admin-dashboard.html`](./ui/admin-dashboard.html) | سایدبار، کارت‌های آماری، نمودار Chart.js، جداول سفارش و موجودی |

---

## 🎨 مشخصات فنی

- **فریمورک:** Bootstrap 5 RTL
- **فونت:** [Vazirmatn](https://fonts.google.com/specimen/Vazirmatn)
- **نمودار:** Chart.js
- **آیکون:** Font Awesome 6
- **تصاویر:** [picsum.photos](https://picsum.photos) (placeholder)
- **زبان:** فارسی (RTL)
- **طراحی:** Responsive — موبایل، تبلت، دسکتاپ
